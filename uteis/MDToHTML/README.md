# Conversor

Converte MD para HTML e permite imprimir o conteudo com um botão no final para usar o imprimir do navegador.

## Regras

- `# ` vira `<h1>`
- `## ` vira `<h2>`
- `### ` vira `<h3>`
- `- ` vira `<li>`
- `> ` vira uma observação estilizada
- Somente blocos de código com ``` são suportados. (pelo menos para java)

---

## Dependencia

- java
